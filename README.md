# Final Project: RPG Character Management System

## รายละเอียดโปรเจกต์
ซอฟต์แวร์นี้เป็นระบบจัดการตัวละคร RPG ที่พัฒนาด้วยภาษา Python 
โดยเน้นการประยุกต์ใช้หลักการ OOP และ SOLID Principles 

### หลักการที่นำมาใช้
1. **OOP (Inheritance & Polymorphism):** ใช้ในการสร้างตัวละครสายอาชีพต่างๆ
2. **SOLID (SRP & DIP):** แยกส่วนการแสดงผลและส่วนการสร้างตัวละครออกจากกัน
3. **Design Pattern:** ใช้ Factory Method ในการสร้าง Object ของตัวละคร

## วิธีการติดตั้งและใช้งาน
1. Clone repository นี้ลงเครื่อง
2. ตรวจสอบเวอร์ชัน Python (แนะนำ 3.10 ขึ้นไป)
3. รันโปรแกรมด้วยคำสั่ง: `python main.py` หรือ กดปุ่ม run python file
